<!-- 주제 선정 이유 및 프로젝트 목적 -->
<div align="center">
  <h2>주제 선정 이유 & 프로젝트 목적</h2>

  <p>
    현재 수능 학습 시장은 강의·교재 등 콘텐츠는 충분하지만 <br/>
    학생이 스스로 공부를 설계·실행·지속할 구조는 부족한 상황입니다. <br/>
    이로 인해 사교육 참여율은 높아졌음에도 <b>노력 대비 성과가 나타나지 않는 비효율이 반복</b>되고 있습니다.<br/>
  </p>
  
  <p>
    설스터디는 이러한 문제의 본질을 <b>학습 설계·관리·피드백 구조의 부재로 정의하고</b> <br/>
    자체 콘텐츠 기반 학습 코칭 시스템을 통해 학생의 공부 방법·실행력·관리 구조를 근본적으로 개선하는 것을 목표로 합니다. <br/>
  </p>
  <p>
    궁극적으로 설스터디는 <b> 학생이 스스로 공부를 설계하고 지속할 수 있도록 돕는 <br/>
    수능 국·영·수 학습 코칭 플랫폼 구축</b>을 지향합니다.
  </p>
